# 👨‍💻 Noor Nabi's Personal Portfolio

Welcome to the repository of my personal portfolio website! This project showcases my skills, experience, and the software development projects I've built. It is designed to be interactive, responsive, and visually appealing.

## 🚀 Live Demo

**[View Live Portfolio](https://mdnoornabi.vercel.app)**

## ✨ Features

- **Home/Hero Section**: A welcoming introduction with dynamic animations.
- **About Me**: A detailed overview of my background, passion for coding, and professional journey.
- **Skills**: A comprehensive showcase of my technical skills.
- **Projects**: Detailed case studies of my work, complete with project descriptions, technologies used, and links to live demos and GitHub repositories. Includes a dynamic project details routing system.
- **Education**: A layout depicting my academic background and qualifications.
- **Contact Form**: A functional contact form powered by **Formspree** allowing visitors to easily get in touch with me directly from the website.
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices.
- **Smooth Animations**: Engaging and modern micro-interactions and scroll animations using **Framer Motion**.

## 🛠️ Tech Stack

This project was built using modern web development technologies:

- **Frontend Framework**: [React](https://react.dev/) (v19)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4) & [DaisyUI](https://daisyui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router](https://reactrouter.com/) (v7)
- **Form Handling**: [Formspree](https://formspree.io/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/), [FontAwesome](https://fontawesome.com/), and [Heroicons](https://heroicons.com/)

## 🏃‍♂️ Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

You will need to have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5173` to view the application in your local environment.

## 📁 Project Structure

```text
src/
├── assets/        # Images, fonts, and static assets
├── Components/    # Reusable React components

├── router/        # React Router configuration
├── Shared/        # Shared components like Navbar and Footer

```
